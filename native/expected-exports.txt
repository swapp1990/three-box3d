# Expected WASM export list for native/dist/box3d.wasm
#
# Ground truth = every `b3bridge_*` function defined in native/bridge.c
# (verified by grep, not by trusting old build logs or docs — the old
# repo's saved build logs/settings listed only 19 EXPORTED_FUNCTIONS
# while the actual shipped WASM exported 25 b3bridge_* functions
# (confirmed here by instantiating the old repo's box3d.wasm directly
# in Node and reading instance.exports — not by trusting secondhand
# counts). This repo's bridge.c adds 6 new wrappers (destroyJoint,
# setAngularVelocity, getAngularVelocity, applyForce, applyTorque,
# setBodyTransform) for a total of 31. This file exists so future
# export-count drift is caught automatically instead of silently
# shipping a stale list again.
#
# Format: one Emscripten EXPORTED_FUNCTIONS name per line (with leading
# underscore), blank lines and lines starting with # are ignored.
# native/scripts/build-wasm.sh derives its -s EXPORTED_FUNCTIONS list
# from this file and also diffs the built .wasm's actual export names
# against it after compiling.
#
